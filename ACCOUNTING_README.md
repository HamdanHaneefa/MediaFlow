# MediaFlow CRM - Accounting Module

A comprehensive accounting and financial management system for media production companies.

## Features

### 1. Expense Entry & Tracking
- **Add Expenses**: Easy form to enter new expenses with project assignment, category selection, date, and receipt upload
- **Filter & Search**: Filter expenses by project, category, status, date range, and submitter
- **Category Management**: Predefined categories including Equipment Rental, Location, Travel, Catering, Crew, Post Production, Marketing, etc.
- **Receipt Management**: Upload and attach receipts or documents for audit trails

### 2. Expense Approval Workflow
- **Status Management**: Draft → Submitted → Approved/Rejected → Paid/Reimbursed
- **Approval Actions**: Managers can approve or reject expenses with feedback
- **Notifications**: Visual indicators for pending approvals
- **Audit Trail**: Track who approved/rejected expenses and when

### 3. Income Management
- **Income Types**: Project payments, deposits, final payments, additional services
- **Status Tracking**: Expected → Received/Overdue/Cancelled
- **Invoice Integration**: Link income entries to invoice numbers
- **Client Association**: Connect income to specific clients and projects

### 4. Financial Dashboard
- **Real-time Metrics**: Total income, expenses, net profit, profit margins
- **Visual Charts**: Monthly trends, expense distribution, project profitability
- **Status Overview**: Pending approvals, expected vs received income
- **Alert System**: Notifications for overdue payments and pending approvals

### 5. Comprehensive Reporting
- **Profit & Loss Reports**: Complete P&L statements with monthly trends
- **Expense Analysis**: Category breakdowns, spending patterns, top categories
- **Income Tracking**: Payment collection rates, average payment times
- **Project Profitability**: Compare income vs expenses by project with profit margins
- **Export Options**: CSV and PDF export capabilities

### 6. Project Integration
- **Project Assignment**: Link expenses and income to specific projects
- **Project Financial Summary**: View complete financial picture per project
- **Profit Margin Calculation**: Automatic calculation of project profitability

## Navigation

Access the Accounting module through the main sidebar navigation. The module is organized into four main tabs:

1. **Dashboard** - Overview of financial status and key metrics
2. **Expenses** - Manage and track all expense entries
3. **Income** - Track expected and received payments
4. **Reports** - Comprehensive financial analysis and reporting

## Getting Started

1. Navigate to `/accounting` in the application
2. Start by adding your first expense using the "Add Expense" button
3. Configure expense categories and approval workflows as needed
4. Add income entries to track expected payments
5. Use the dashboard to monitor financial performance
6. Generate reports for financial analysis and export
